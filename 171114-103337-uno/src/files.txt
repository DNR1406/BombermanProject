In dit bestand staat over elk bestand in deze map wat er in staat, waar het over gaat. 
Als je even niet meer weet waar een bestand voor is, kijk dan even hier, als je een nieuw bestand aanmaakt, voeg hem dan even toe.


~
CommunicationIR.hpp     Header file voor communicatie
CommunicationIR.cpp     Alle functies die te maken hebben met communicatie

~
Navigation.hpp          De header file voor het navigatie menu
Navigation.cpp          Alle functies om te tekenen, schemren te maken

~
NavigationScreen.hpp    Header file voor het scherm
NavigationScreen.cpp    Alle specefieke functies om dingen op het scherm te zetten, knoppen, headers e.d.

~
ArduinoNunchuck.hpp     Header file voor de nunchuck
ArduinoNunchuck.cpp     Alle functies

~
ArduinoNunchuck.hpp     Header file voor nunchuck.cpp.
ArduinoNunchuck.cpp     De functies om de gegevens van de nunchuck op te halen.

~
Bomb.hpp 				Header file voor Bomb.cpp
Bomb.cpp				Functies die te maken hebben met bommen

~
Drawings.hpp            Header voor de tekeningen
Drawings.cpp            Alle afbeeldingen, functies om afbeeldingen te tekenen

~
Functions.hpp           Header file voor alle algemene functies
Functies.cpp            algemene functies, timers, inits, ISR e.d.

~
GameEngine.hpp			Header file voor GameEngine.h 
GameEngine.cpp			Alle functies die het spel bepalen komen hierin te staan (levels, players, highscores e.d.)

~
Glabals.hpp             Bestand voor alle globale variablen die over het hele project te gebruiken zijn.

~
Main.cpp                main programma.

~
Map.hpp 				Header file voor Map.cpp
Map.cpp					De functies die de map tekenen en de locaties van alle objecten worden in deze klasse gemaakt

~
PlayerMovement.hpp 		Header file voor Player.cpp
PlayerMovement.cpp 		Alle functies die bij een speler horen komen in deze klasse (levens, xlocatie, ylocatie enz.)
